LFG

The problem

Matchmaking is deliberately poor in Destiny

People don't seem to like that and want services like this.

Unique Selling Point

An automatic way to join groups

Don't have to scan posts or chat

Direct notifications

Application Flow

10. Log in with your game name, platform and role

20. Select activities that you are interested in

30. The backend creates groups from interested players

40. Players can accept or decline group invite (decline GOTO 20)

50. Declining players are replaced, continues until all accept

60. The group is created in game by players

Making it Consistent

Everything runs inside Docker containers

Improves reproducibility and distribution

Docker-Compose allows orchestration of multiple connected docker containers

Only the Java Socket Server is deployed in a container

They are used for all components during development

Supporting Technology - Behave

Multi-User interaction is complicated and hard to test

Some of the logic in LFG depends on multiple people interacting

Behave is used to test these use cases and allows the tests to be clear and comprehensive
